# 1D CNN with BLSTM for Eye Movement Classification

Automated classification of fixations, saccades, and smooth pursuits using a 1D Convolutional Neural Network with Bidirectional Long Short-Term Memory.

## Paper

**1D CNN with BLSTM for automated classification of fixations, saccades, and smooth pursuits**

Mikhail Startsev, Ioannis Agtzidis, Michael Dorr

Behavior Research Methods, 2019

The original code has been slightly modified to align with our data.
